# Beam Bot
- This is a simple beam bot I made

# Setup
- Python required !
- https://www.python.org/
```python
pip install -r requirements.txt 
```
- Settings:
```
  "token": "The bot token",
  "Webhook": "Here the webhook url",
  "Prefix": "Here your prefix, e.g. !"
  "Company": "The title of the embed"
```
