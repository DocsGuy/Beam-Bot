# Beam Bot
- 这是我做的一个简单的光束机器人

# 设置
- 需要Python!
- https://www.python.org/
```python
pip install -r requirements.txt 
```
- 设置。
```
  "token": "机器人的token"。
  "Webhook："这里是Webhook网址"。
  "前缀"："这里是你的前缀，例如！"
  "公司"："被嵌入的标题"。
```

# 功能
- 向webhook集成cookie发送者
- 半cookie检查器
- 假饼干的事
# 它能做什么？
```
当你输入!"cookie"(Roblox cookie)时。
它创建一个假的检查器并等待5秒钟，然后发送一个消息（如果消息中有一个cookie）。
并说"✅成功! 您的cookie将很快发送给您"。
然后在2秒后发送一个生成的cookie
```
